# Free-online-tools

A personal collection of free, browser-based tools available at [civvy.tech](https://civvy.tech).

No login. No ads. No installation. Just open and use.

## About

I built [civvy.tech](https://civvy.tech) as a side project to collect tools I kept needing but couldn't find in a clean, distraction-free form.

Everything runs directly in the browser. No data leaves your device.

Learn more on the [About page](https://civvy.tech/about).
